# 📊 Financial Admin Dashboard

A clean, responsive, and modern admin dashboard template featuring a dark-themed user interface designed for financial tracking and user activity management. Built with a mobile-first approach to ensure seamless operation across all device viewports.

## ✨ Key Features

* **Dynamic Accordion Sidebar:** A multi-level, collapsible navigation menu for organizing dashboard activities, deposits, and withdrawals cleanly without cluttering the screen.
* **Responsive Statistics Grid:** A responsive card-based layout that automatically scales to display user balance, bonuses, and account status beautifully on mobile, tablet, and desktop screens.
* **Actionable Alert System:** Built-in dismissible alerts for crucial user notifications, such as account verification requirements and funding status.
* **User Profile Navigation:** An integrated navbar featuring quick-action buttons (Add Funds/Withdraw) alongside a dynamic user profile dropdown menu.

---

## 🛠️ Technologies Used

* **HTML5 & CSS3:** For semantic structure and custom styling.
* **Bootstrap 5:** Leveraging modern grid systems, flexbox utilities, and responsive layout structures.
* **FontAwesome 6:** For clean, scalable, and intuitive vector iconography across the interface.
